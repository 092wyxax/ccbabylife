## 3. 收益分配

### 3.1 月收益計算
```
營業額（不含稅）
- 商品成本
- 國際運費實際分攤
- 平台費用（綠界、LINE、伺服器、AI API）
- 公司稅費
= 月毛利
```

### 3.2 分配優先順序

1. **每月先存 30% 進「公司營運準備金」**，到達 NT$500,000 上限後此項可降為 10%
2. 剩餘部分依以下分配：

| 項目 | % | 給誰 | 備註 |
|---|---|---|---|
| 系統開發回饋金 | 待定 | 先生 | 補貼開發時間 |
| 內容 / 客服回饋金 | 待定 | 太太 | 補貼日常營運時間 |
| 採購回饋金 | 待定 | 朋友 | 含日本實地時間 |
| 出資按比例分紅 | 餘 | 三人按出資比例 | 與職務薪資分開計算 |

> ⚠ 真正的數字三人協商後填入，並以律師合約為準。

## 4. 工作分工細節

### 4.1 商品上架流程
1. 朋友提案（日本市場觀察 / 廠商接洽 / 市場情報系統推薦）
2. 太太評估（目標客群匹配度 / 心得能不能寫）
3. 先生法規檢核（對照 LEGAL_GUIDE.md）
4. 三方同意 → 太太建檔上架

### 4.2 單筆訂單流程
- 客戶下單：自動 → 太太收到通知
- 日本下單：朋友執行
- 集運：先生協調物流商
- 報關：先生
- 出貨：先生（重物）+ 太太（輕物 / 美感包裝）
- 客服：太太主辦，先生支援技術問題

### 4.3 重大決策
任何以下事項需**三方書面同意**：
- 上架紅燈或黃燈品項
- 大筆採購（單批 > NT$500,000）
- 通路擴張（蝦皮、Pinkoi 等）
- 雇用第一個外人
- SaaS 副業切割
- 結束營業

## 5. 財務透明

- 每月 1 號公開上月財報（先生負責產生）
- 公司戶頭三人皆可查（不限提領）
- 大筆支出（> NT$50,000）需事前 LINE 告知
- 薪資 / 分紅按月發放

---

## 6. 退場機制

### 6.1 一方退出
- 提前 60 天書面通知
- 評價公司價值（可請外部會計師）
- 退出方持股由其他兩人按比例承接（或公司回購）
- 退出後 6 個月內不得從事直接競爭業務（待協商）

### 6.2 結束營業
- 三方一致決議
- 清算庫存、結算稅務
- 剩餘資產按出資比例分配
- 客戶資料依個資法處理（保留 / 刪除）

### 6.3 一方逝世 / 失能
- 由其遺屬 / 監護人代位繼承股權
- 公司營運不受影響
- 必要時其他合夥人可協商買回繼承股權

## 7. 衝突處理

- 第一輪：私下協商（24 小時內）
- 第二輪：寫成文字，三方都看過再回應（72 小時內）
- 第三輪：找一位中立第三方（律師、會計師）協調
- 最後：依股權多數決（但需符合 §4.3 重大決策原則）

## 9. 育兒衝突

特別條款：太太是主要照顧者，本店是「副業」、不是優先於育兒的事業。

- 我家娃生病、學校事務、健康檢查 → 任何工作可暫停
- 重要訂單期遇此狀況 → 朋友與先生臨時補位
- 公司不要求太太「全力投入」、太太也不需內疚於工作量
- 客戶因此延遲收貨 → 主動道歉 + 補償購物金，不甩鍋給太太
