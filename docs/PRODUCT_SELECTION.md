# 30 件選品清單

> 起步 30 件 SKU 的清單。實際內容請太太 + 朋友填入。
> 格式約定：每件商品有「品名／品牌／日本通路／日幣售價／重量／品類／月齡／法規檢核狀態／我家娃心得」。

## 4. 選品守則

### 4.1 必要條件
- 通過法規檢核（`docs/LEGAL_GUIDE.md`）
- 有日方品牌 / 通路規格說明（不買來路不明貨）
- 朋友能穩定取貨（熱門品有缺貨風險，標 `stock_type: limited`）

### 4.2 加分條件
- 太太 / 我家娃 / 朋友家貓狗親身試過
- 在台灣**買不到**或**很難買到**
- 有清楚的差異化（材質、設計、品牌故事）

### 4.3 排除
- 純粹「日本比較便宜」的商品（毛利低、競爭高）
- 容易跑單的高價爭議品（無品牌頂級美妝、限量奢侈品）
- 任何法規黃燈以上的（除非朋友 + 先生雙方核可）
