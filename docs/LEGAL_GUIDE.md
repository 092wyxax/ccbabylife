# 法規地雷清單（紅燈品項）

> 「不賣需查驗登記商品」是日系選物店的核心差異化。
> 本清單列出**絕對不代購**的品類，以及需審慎評估的品類。
> 商品上架前須對照本清單；後台 `products.legal_check_passed` 需勾選確認。

---

## 1. 紅燈：絕對不代購

### 1.1 母嬰類

| 品類 | 法規 | 說明 |
|---|---|---|
| 嬰兒奶粉 | 食安法、嬰兒與較大嬰兒配方食品查驗登記準則 | 需查驗登記，違者重罰 |
| 嬰幼兒副食品（米精、麥精、嬰兒食品罐頭） | 食安法 | 需查驗登記 |
| 嬰兒專用補充劑（DHA、益生菌） | 健康食品管理法、藥事法 | 視成分可能落入藥品 |
| 嬰兒退熱貼、退熱貼劑 | 醫療器材管理法 | 需衛福部許可 |
| 處方藥、感冒藥、止痛藥（即便無處方箋日本可購） | 藥事法 | **任何藥品**禁止代購 |
| 醫療器材（額溫槍、體溫計、血氧機等） | 醫療器材管理法 | 需查驗登記 |
| 含藥化妝品（美白、防曬係數標示、抗痘） | 化妝品衛生安全管理法 | 需備查 |

### 1.2 寵物類

| 品類 | 法規 | 說明 |
|---|---|---|
| 寵物處方食品（c/d、k/d、Royal Canin 處方系列） | 藥事法、動物用藥品管理法 | 需獸醫處方 |
| 寵物用藥（驅蟲、心絲蟲、皮膚藥） | 動物用藥品管理法 | 需查驗登記 |
| 寵物保健品（添加 CBD、特定營養素） | 視成分而定 | 案例多模糊地帶，避免 |
| 含肉類寵物食品 | 動物傳染病防治條例 | 進口需檢疫，個人 6 kg 以下例外但商業不適用 |
| 含鮮乳製品的寵物食品 | 同上 | 同上 |

### 1.3 食品類

| 品類 | 法規 |
|---|---|
| 一般食品（餅乾、糖果、飲料、調味料）| 食安法 — 商業進口需查驗登記 |
| 健康食品（標 13 項保健功效） | 健康食品管理法 |
| 含酒精商品 | 菸酒管理法 |

### 1.4 其他

| 品類 | 法規 |
|---|---|
| 武器、刀械（料理刀含部分日本和包丁）| 槍砲彈藥刀械管制條例 |
| 動植物標本、毛皮製品 | 野生動物保育法、CITES |
| 仿冒品、未授權聯名 | 商標法 |
| 成人用品、未送審玩具 | 商品檢驗法 |

## 2. 黃燈：需個案評估

### 2.1 嬰幼兒玩具
- **3 歲以下嬰幼兒玩具**需經商品檢驗 BSMI 認證才可商業販售
- 解法：上架前查日本商品是否有 BSMI 對應驗證；若無，標示「個人收藏品」並只接受訂金制（朋友親自帶回，不走報關通路）— 但**仍有風險**

### 2.2 嬰幼兒用品（奶瓶、餐具、安撫巾）
- 食品接觸材質需符合 CNS 1009 等標準
- 解法：選日本同等以上規格的品牌（Pigeon、Combi、Richell 等大品牌通常合規）

### 2.3 寵物用品（牽繩、項圈、玩具）
- 一般無法規限制，可代購
- 含電子元件（自動逗貓棒、噴水機）需 BSMI

### 2.4 化妝品 / 保養品（不含藥）
- 一般化妝品需備查（業者備查制）
- 母嬰專用乳液若不主張療效，視同一般化妝品
- 朋友建議：母嬰選物店暫不碰化妝品，避免複雜度

## 3. 綠燈：可代購

| 品類 | 備註 |
|---|---|
| 嬰兒服飾、童裝 | 注意安全標準（拉繩、釦子大小） |
| 嬰兒用品（棉布、紗布、固齒器）| 需 CNS 認證的選大廠 |
| 寵物乾糧（無含肉檢疫疑慮的，但較少）| 多數日本乾糧含肉，實際多落黃燈 |
| 寵物外出用品（牽繩、外出包） | 安全 |
| 寵物玩具（純布、純橡膠）| 安全 |
| 母嬰選物（包巾、推車配件、收納用品）| 安全 |

## 4. 法規誠信宣告（公開於 `/about`）

> 我們堅持只販售綠燈品項。
> 我們**不代購**任何需查驗登記、需處方、進口檢疫不適用商業通路的商品。
> 即使客戶詢問，我們會婉拒並說明原因。
> 這是本店與其他日本代購最大的差異 — 我們選擇難走但合法的路。

---

## 5. 上架檢核 SOP

每件商品上架前，太太需走完此檢核：

- [ ] 對照本清單，確認非紅燈
- [ ] 若黃燈，朋友 + 先生雙方核可
- [ ] 商品成分／材質有日方規格說明
- [ ] 後台 `products.legal_check_passed = true`
- [ ] 商品頁不主張任何療效、保健功效、醫療效果
- [ ] 標示來源：「日本平行輸入」、「個人選物」（不主張代理）

---

## 6. 客訴與舉報處理

- 收到「為什麼不賣 X？」 → 引導至 `/about` 法規誠信宣告
- 收到他店違法舉報邀請 → 不參與，專注自家
- 收到主管機關函詢 → 立即聯繫律師，先生主處理
