# 品牌指南

> 本文件規範品牌的視覺、語氣、調性，確保前台、LINE、IG/Threads、客服 LINE 訊息一致。
> 主要由太太維護；先生在 UI/Email 模板實作時參考。

---

## 1. 品牌核心

### 1.1 一句話定位
> 「1 歲娃媽親身試用、嚴選日系好物、不賣需查驗登記商品」

### 1.2 三個關鍵字
- **真實**（媽媽本人 + 真實使用心得）
- **嚴選**（30 件起步，寧少不濫）
- **誠信**（法規地雷不碰）

### 1.3 我們不是
- 不是「全品項日本代購」
- 不是「最便宜」
- 不是「網紅選物」

## 3. 語氣（重要）

### 3.1 自稱
- 對客戶：「我們」「小編」「娃媽」（看情境）
- 不用「本店」「敝公司」（過度疏離）

### 3.2 對客戶稱呼
- 預設：「妳」「妳們」（女性客群為主）
- 男性家長：「您」（保留禮貌空間）

### 3.3 三條語氣原則

#### 原則 1：誠實，不造作
- ✅「我家娃用這個三個月，最大優點是 X，缺點是 Y，我覺得 Z 才會考慮購買」
- ❌「100% 媽媽推薦！神級好物！」

#### 原則 2：知識，不販售
- ✅「日本嬰兒衣服尺寸 80 通常對應台灣 1 歲，但我家娃比較高，我都買 90」
- ❌「立刻購買享受優惠！」

#### 原則 3：邊界感
- 不過度親密、不裝熟
- 客訴要直接道歉，但不卑微

## 4. 寫作範本

### 4.1 商品頁「我家娃使用心得」（差異化武器 #1）

**結構**：
1. **背景**：哪個月齡 / 場景開始用
2. **優點**：具體、可驗證的細節（不空泛）
3. **缺點**：誠實（這是信任建立關鍵）
4. **適合誰／不適合誰**

**範例**：
> 我家娃從 8 個月開始用這個固齒器。
>
> 優點是矽膠軟硬適中、表面有起伏紋路，娃會自己拿著磨牙不哭。日本 Combi 出的，有 BSMI 標誌（編號 R12345），所以我敢給她直接咬。
>
> 缺點是吊環設計，繫在推車上扣具會稍鬆，需要自己加個小束帶。
>
> **適合**：8–18 個月開始長牙、會抓握的寶寶。
> **不適合**：6 個月以下還不會穩抓的，會掉地上要常清洗。

### 4.2 LINE 加好友歡迎詞

```
妳好，這裡是日系選物店～

我們是一家媽媽親選日系母嬰／寵物用品的小店。每週日截單、週一日本下單、約 10–14 天到貨。

⏤
你可以這樣使用我：
・直接貼日本商品 URL，30 秒自動報價
・問訂單狀態：訂單編號 + 「狀態」
・有問題會在 24 小時內回覆（週一～五）
⏤

最新選物：請看官網 → [URL]
```

> 視覺上避免大量 emoji，用「⏤」這類分隔字元更日系。

## 5. IG / Threads 風格（太太主導）

### 5.1 圖片
- 自然光、情境感
- 寶寶／寵物正面入鏡（隱私允許）— 我家娃出鏡是品牌資產
- 商品平拍要乾淨，但不無聊（加生活元素）

### 5.2 文案結構（IG）
- 第一行：勾子（一句話）
- 中段：3–5 句心得
- 結尾：「想知道更多請看官網（連結在 bio）」
- Hashtag：#日系選物 #媽媽真心話 #日本代購（精簡 5–10 個）

### 5.3 Threads 風格
- 更短、更口語、更敢
- 可丟議題（「為什麼日本嬰兒服飾比較貴？」）
- 不主動推商品（推太多會疲乏）

## 6. 客服話術

### 6.1 拒絕代購紅燈品

```
妳好，這款商品屬於需查驗登記的醫療器材／處方藥／嬰兒奶粉，
依台灣法規我們無法代購販售。
這是我們選物店「不賣需查驗登記商品」的核心原則之一，
還請見諒～

可以推薦給妳同類安全的選物：[商品連結]
```

### 6.2 客訴道歉

```
不好意思讓妳有這樣的體驗。
能請妳將收到的狀況拍照給我嗎？
我們確認後會立刻[退貨／換貨／退款]，並補償購物金 NT$XXX。
```

不要用「親」「哈囉～」這類過度親密、或用大量驚嘆號／emoji。
